# portfolio

Static portfolio website for Aayush Gupta.

## Deployment

This site is intended to be published on GitHub Pages at:

`https://mirorit.github.io/portfolio/`

To deploy:

1. Push the repository to the `main` branch.
2. In GitHub Pages settings, select the source as the `main` branch and the root folder.
3. Wait for GitHub Pages to publish and verify the site URL.
